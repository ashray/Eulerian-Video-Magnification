Project

Project Abstract

This project aims to develop algorithms to extract out subtle changes in a time-dependent data set and amplify them. To begin with, the data set can be considered as videos(2D data at each temporal resolution) but scope of the project can be modified to deal with different dimensional datasets at each time instant.

Technical Details

Below we have explained the significant steps of the algorithm.

  • We first consider each of the frame of the video independently
  • We choose a suitable colour space in which we want to work, This could depend on the specific application that we are dealing with, though in the paper, authors have used NTSC colour space for further operations.
  • For each of the color(or spectrum level for hyper spectral images), we build a Laplacian pyramid. Note the Laplacian Pyramid is build of the NTSC domain image, not the RGB image.
  • We define the lowpass filter to have the Laplacian Pyramid values. Note that the Laplacian Pyramids are used to smooth in the spatial domain.
  • We consider the next frame, and perform the similar laplacian pyramid calculation on it.
  • The value of laplacian pyramids of subsequent frames is used to perform the temporal filtering of the signal. The exact method of temporal filtering could vary with application, for ex. we could use a butterworth filter, an IIR filter etc.
  • The above method will give us the lower and upper frequency bounds. We calculate the difference of them to get the frequencies that we want to filter(magnify or suppress).
  • Given the selected frequencies, we selectively magnify them based on user preference. The paper has used a linear magnification till a certain frequency and constant magnification after a certain threshold frequency as the choice, though this can be left as a user defined parameter.
